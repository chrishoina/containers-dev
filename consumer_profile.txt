## Describing Costs, Products, and Sales Tables

### SQL Used to Describe Tables
DESCRIBE SH.Costs
DESCRIBE SH.Products
DESCRIBE SH.Sales

### Summary of Tables
The Costs table stores information about the cost of products over time. 
The Products table contains detailed information about each product, including its name, description, category, and pricing. 
The Sales table records sales transactions, including the product sold, customer, time of sale, and amount sold.

### SQL Used to Retrieve Top 20 Products by Margin
SELECT /* claude-3.5-sonnet */ s.prod_id, SUM(s.amount_sold) AS total_sales_revenue FROM SH.Sales s JOIN SH.Costs c ON s.prod_id = c.prod_id AND s.time_id = c.time_id GROUP BY s.prod_id ORDER BY (SUM(s.amount_sold) - SUM(c.unit_cost * s.quantity_sold)) / SUM(s.amount_sold) DESC, SUM(s.amount_sold) ASC FETCH FIRST 20 ROWS ONLY

## Describing Customers, Supplementary_Demographics, Products, and Countries Tables

### SQL Used to Describe Tables
DESCRIBE SH.Customers
DESCRIBE SH.Supplementary_Demographics
DESCRIBE SH.Products
DESCRIBE SH.Countries

### Summary of Tables
The Customers table stores information about individual customers, including their demographic details. 
The Supplementary_Demographics table provides additional demographic information about customers. 
The Products table contains detailed information about each product, including its name, description, category, and pricing. 
The Countries table stores information about different countries, including their ISO codes and names.
